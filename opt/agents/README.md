# Agents

Türkçe denetimi/düzeltmesi yapan muhtelif araçlar.

## Skills

### turkish-check

Türkçe içeriklerde dilbilgisi, yazım ve üslup denetimi yapar.

**Tetikleyiciler**: `türkçe denetimi`, `türkçe denetle`, `review turkish`, `check turkish`

**Başvuru dosyaları**:

- `confusables.tsv`: Sık karıştırılan kelimeler ve anlamları
- `misspellings.tsv`: Yaygın yazım hataları ve doğru yazılışları
